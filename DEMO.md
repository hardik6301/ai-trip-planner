# Travora — 60-second demo script

Use this for interviews / resume walkthroughs.  
Live: https://travora-ai-app.vercel.app

## Setup before recording

1. Sign in as a **Pro** user (or set `is_pro = true` in Supabase `profiles`).
2. Have one saved trip ready (e.g. Bangkok).
3. Use desktop Chrome/Edge; dark mode optional.

## Script (≈60 seconds)

| Time | Say / do |
|------|----------|
| 0:00 | “Travora is an AI trip planner I built with Next.js, Supabase, Gemini, and Razorpay.” |
| 0:05 | Home → type destination → generate. Show loading skeleton. |
| 0:15 | Results: real destination hero, weather/currency, day cards, Maps link. |
| 0:25 | Open AI chat FAB → “Make Day 2 more relaxing” → day highlights update. |
| 0:40 | Budget card → Calendar export **or** Offline Pack **or** Expense Tracker. |
| 0:50 | “Free users are capped at 5 trips and 3 regenerations — limits are enforced in API routes.” |
| 0:55 | Close on live URL + freemium + payments story. |

## Resume one-liner

Full-stack AI trip planner with Gemini itineraries, freemium gating, Razorpay Pro unlock, chat-based editing, expense analytics, and calendar/offline exports — deployed on Vercel.
